# Resilience

## Installation :


```
git clone https://github.com/lleoooell/resilience.git
cd resilience
npm install
node server.js
```

## View app :

```
app is living now on your server:3000
```

## Deploy for production with pm2 :

```
git clone https://github.com/lleoooell/resilience.git
cd resilience
npm install
pm2 start server.js --name resilience
```
