Résilience

Capacité d'un individu à supporter psychiquement les épreuves de la vie. Capacité qui lui permet de rebondir, de prendre un nouveau départ après un traumatisme

Parce que la lutte contre le Cornavirus est une affaire collective et que nous avons tous un rôle à jouer en faisant preuve de solidarité.

Un masque de plus pour un professionnel c’est une demi journée de plus pour assurer son service. Chaque jour compte !

Mobilisons nous !

Peut être avez vous un stock de matériel à la maison ou dans votre entreprise qui peut aider ceux qui luttent au quotidien pour assurer la santé, l’alimentation et la logistique indispensable pour faire fonctionner le pays en temps de crise.

Via cette plateforme vous pouvez faire un don autour de vous ou faire un appel aux dons de matériel qui pourrait vous aider si vous êtes un professionel.

Données et Vie privée

Cette application à été développée en Open Source par Léo Casagrande et Olivier Veinand, uniquement dans le but de voir naitre des chaines de solidarité locales afin de protéger les professionnels qui luttent tous les jours pendant que nous sommes confinés. Toute utilisation commerciale est exclue et proscrite

Les données personnelles ne sont pas stockées et ne seront aucunement utilisées à d'autres fins que celle ci.

Amis Développeurs

N’hésitez pas à télécharger le code source ici pour déployer cette application dans votre propre région. Il s'agit d'une simple application nodejs qui se lance en quelques commandes : 

$ git clone repo.git
$ cd repoFolder
$ npm install
$ node server.js
